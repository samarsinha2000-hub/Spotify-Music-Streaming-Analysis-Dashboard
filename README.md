# Spotify-Music-Streaming-Analysis-Dashboard
1️⃣ Project Title / Headline

Spotify Music Streaming Analysis Dashboard | Power BI

2️⃣ Short Description / Purpose

Developed an interactive dashboard to analyze Spotify music streaming data and understand listening trends, song popularity, artist performance, and user preferences. 
The project helps identify top-performing tracks and artists, analyze streaming behavior over time, and support data-driven insights for music consumption patterns.

3️⃣ Tech Stack
1. Power BI – Dashboard development and reporting
2. Power Query – Data cleaning and transformation
3. DAX – Creation of calculated measures and KPIs
4. Excel / CSV – Data source

4️⃣ Features / Highlights
1. Built an interactive Power BI dashboard to analyze song streams, artist performance, and popularity metrics
2. Cleaned and transformed raw Spotify data using Power Query to ensure accurate and consistent reporting
3. Created DAX measures for Total Streams, Average Popularity Score, Track Count, and key KPIs
4. Analyzed top artists, top tracks, and genres based on streaming performance
5. Performed time-based trend analysis to identify changes in listening behavior over different periods
6. Designed clear and engaging data visualizations to compare performance and deliver actionable insights
